# Rock Paper Scissors

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. The player competes against the computer, and the result is displayed instantly after each round.

## Live Demo

View the project here:
[https://Praneeth5607.github.io/frontend-projects/Rock-Paper-Scissors/](https://Praneeth5607.github.io/frontend-projects/Rock-Paper-Scissors/)

## Features

* Play against the computer
* Random computer choices every round
* Instant win/lose/draw result display
* Clean and responsive interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## Project Structure

```
Rock-Paper-Scissors/
├── index.html
├── style.css
└── script.js
```

## Purpose

This project was created as a beginner-friendly exercise to practice JavaScript logic, DOM manipulation, and basic UI styling.
