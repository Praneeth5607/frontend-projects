# Frontend Profile Card

A modern and responsive profile card built using **HTML, CSS, and JavaScript**.
This project showcases a personal introduction, skills, social links, and a call-to-action button in a clean card layout.

## Live Demo

View the project here:
[https://Praneeth5607.github.io/frontend-projects/Profile%20Card/
](https://Praneeth5607.github.io/frontend-projects/Profile%20Card/
)


## Features

* Profile image with circular layout
* Name and role section
* Short professional tagline
* Skills display
* Social media icons (GitHub, LinkedIn, Email)
* Interactive “View Projects” button using JavaScript
* Responsive and minimal design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome (for icons)

## Folder Structure

```
profile-card/
│── index.html
│── style.css
│── script.js
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in a browser.

## Customization

* Replace the profile image inside the `images` folder.
* Edit the name, role, and skills in `index.html`.
* Modify styles in `style.css`.
* Update button actions or interactions in `script.js`.

## Purpose

This project was created as a beginner frontend exercise to practice layout, styling, and basic interactivity using HTML, CSS, and JavaScript.
